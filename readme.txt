== Description ==

This widget shows global virus and malware activity level with latest malware threats. It is useful to let your visitors know how active viruses are because, if this level is high, visitors will know that they need to keep attention on their system security. This plugin is perfect for computer repair shops, computer technicians and similar service providers who what to show their customers global computer virus activity.
The plugin uses RSS feeds from [Virus Activity information center](https://virusactivity.com/) to show and update threat level data and the list of latest threats. You can configure how many cyber threats you want to display in widget configuration.

= Features list =

* Multi-language (25+ languages supported)
* Displays activity level and last update date
* Customizable widget title
* Customizable number of the latest threats. Up to 10 items
* Daily updated


== Installation ==

This section describes how to install the plugin and how to get it working.

= From your WordPress dashboard =

1. Visit 'Plugins' > 'Add New'
2. Search for 'Virus Activity' and install it
3. Activate **Virus Activity** plugin from your Plugin’s page

= From WordPress.org plugin repository =

1. Download **Virus Activity** zip file and unzip it
2. Upload the entire `virus-activity` folder to the `/wp-content/plugins/` directory using your favorite method (FTP, SCP, etc...)
3. Activate **Virus Activity** plugin from your Plugins page

= Once Activated =

1. Go to widget’s page and drag and drop the newly-added Virus Activity to any of your sidebars
2. Configure the widget options
